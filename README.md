# Gliese
Front-end starter kit using [Sass](http://sass-lang.com/), [Webpack](http://webpack.github.io/) with Babel, and using NPM as task-runner.

## Startup
- ```npm install```
- Change path in ```system/config.php```
- Change project data in ```package.json```
- ```npm start```
